AWS API Gateway Path Replacer GUI

Preparing IAM user

  • Login to the AWS console and open IAM console.
  • Select Users on left pain and click "Add user" button.
  • Input User name you want.
    Access type: Choose(check) "Programmatic access".
    Then click "Next: Permissions" button beneath.
  • Select "Attach existing policies directly".
    And choice "AmazonAPIGatewayAdministrator".
    Then click "Next: Tags" button beneath.
  • Click "Next: Review" button beneath and "Create User".
  • Copy your Access key ID and Secret access key for next process.

Preparing AWS credential

  • Open [Your user directory]/.aws/credentials file on your local machine.
    * If there is no .aws/credentials file then:
    - create ".aws" directory
    - create "credentials" file
  • Append text below to your credentials file. And save it.
    [aws_apigw_path_replacer]
aws_access_key_id=xxxx   ... your Access key ID
aws_secret_access_key=yyyy   ... your Secret access key

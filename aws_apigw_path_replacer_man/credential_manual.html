AWS API Gateway Path Replacer GUI

Preparing AWS creadential

  • Open [Your user directory]/.aws/credentials file on your local machine.
    * If there is no .aws/credentials file then:
    - create ".aws" directory
    - create "credentials" file
  • Append text below to your credentials file. And save it.
    	[aws_apigw_path_replacer]
	aws_access_key_id=xxxx   ... your Access key
	aws_secret_access_key=yyyy   ... your Secret
